# esp8266-monitoring
Async websockets client that sends sensor data to vps

# Current sensors
 - Pin `D5`: Flame (`HW-491` / `KY-026`)
 - Pin `D6`: Temperature (`HW-506` / `Ds18b20` / `KY-001`)
